What do we cover?

 
{% thumbnail "http://www.mbasql.com/static/images/lessons-start.png" "100" crop="center" format="PNG" as im %} {% endthumbnail %}
Introduction we will start by learning how all queries are structured and write our first query .
{% thumbnail "http://www.mbasql.com/static/images/lessons-aggregates.png" "100" crop="center" format="PNG" as im %} {% endthumbnail %}
Aggregation we will learn how we can transform, aggregate, group, and sort data we extract.
{% thumbnail "http://www.mbasql.com/static/images/lessons-filtering.png" "100" crop="center" format="PNG" as im %} {% endthumbnail %}
Filtering we will learn how to include and exclude data based on different criteria we set
{% thumbnail "http://www.mbasql.com/static/images/lessons-advanced.png" "100" crop="center" format="PNG" as im %} {% endthumbnail %}
Advanced we will learn how to join data from different tables, use query results as tables, and handle timeseries data.
 

Our Method

 

 
{% thumbnail "http://www.mbasql.com/static/images/002-visibility.png" "150" crop="center" format="PNG" as im %} {% endthumbnail %}

Watch

Every lesson starts with an introduction to a new concept in SQL where we will cover the theoretical aspects of a specific functionality.

{% thumbnail "http://www.mbasql.com/static/images/004-pacman.png" "150" crop="center" format="PNG" as im %} {% endthumbnail %}

Play

You will then put on the hat of a retail store manager and be invited to think about how each concept can be used in a business setting.

{% thumbnail "http://www.mbasql.com/static/images/003-compose.png" "150" crop="center" format="PNG" as im %} {% endthumbnail %}

Do

Once we have completed an example, you will be challenged with a problem you must solve. We will then present the solution and discuss.
